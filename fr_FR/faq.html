FAQ - Foire Aux Questions

Général

Où pourrais-je poser ma question ?

Si vous ne trouvez pas votre réponse dans cette FAQ ou dans le manuel, vous pouvez toujours demander à ici .

La navigation

Comment ouvrir rapidement un élément sur une nouvelle vue ?

Le plus rapide est de cliquer avec le milieu sur l'élément.

Texte

Comment modifier la largeur de la zone de texte principale ?

Sur le côté droit, dans la boîte à outils "Modifier", cliquez sur l'icône "A" avec la flèche vers le haut. Un nouveau menu doit apparaître.

Développement

Où puis-je trouver le référentiel Skribisto ?

Le dépôt GitHub est ici .

Comment puis-je aider à traduire ?

Les instructions complètes sont ici .

À raccourcir : traduisez ici (c'est gratuit) ou créez un problème ici si votre langue est manquante.

Comment pourrais-je contribuer au code ?

Tout d'abord, avant d'écrire un changement, vous pouvez peut-être créer un problème décrivant votre modification, alors on peut en discuter et ainsi je peux vous guider ?

Après cela, vous pouvez forker le référentiel, faire votre changez, puis créez une demande Push de votre référentiel vers le mien.
